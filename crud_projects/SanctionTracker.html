Sanction Tracker

Input provider's details in the input fields

After you input the details, the system will notify that you successfully created a new record

You can see the records that you created in the "Entries" Menu at the left sidebar

You can search any word in the search box

You can also search between the start and end date of the day you created the record

In the "Entries", you can see the full details of the provider.

You can update specific Record

You can delete specific Record by clicking delete and type "delete" for confirmation.

You can Delete all records, but in order to do that, you need additional steps to confirm the deletion

After confirming the deletion, all the records are being erased

I use PHPMyAdmin to create SQL tables for the Provider data and for the user to log in to the system.
