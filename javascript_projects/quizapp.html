Aimed to make a fun and user-friendly platform for quizzes. The goal was to let users take quizzes, manage questions and answers easily, and make everything look good.

Getting started, I needed a way to handle questions and answers. I used an array of objects in JavaScript to store them, making it easy to show and change them on the screen as needed. This was the foundation for everything that came next.

One big challenge was saving and loading data. I found out about localStorage which let me save data in the browser. This meant users could leave and come back without losing their progress.

As I got into the details, I had to make sure users could click buttons and select answers. Using JavaScript, I created buttons for questions and made sure the answers worked properly when clicked. This made the quiz interactive and engaging.

One important part was making everything look good. CSS came in handy here. I used it to arrange elements neatly on the screen. I also added nice colors and styles to buttons, making the app visually appealing.

Another interesting challenge was letting users delete questions. With JavaScript, I added a way for users to remove questions they didn't want. This helped keep things tidy.

But not everything went smoothly. Centering elements and making buttons look just right turned out to be trickier than I thought. With some help from CSS, I managed to make things look good and work well.

Managing different parts of the app wasn't easy either. I created separate files for different functions, making sure each piece worked together without getting confusing.

Of course, there were bumps along the way. Fixing errors and making things work as expected took some time. I used tools like console logs and checked my code carefully to find and solve problems.

In the end, building the quiz app was like going on a journey. It taught me a lot about using HTML, CSS, and JavaScript to create fun and practical things online. From making buttons work to saving data, this adventure showed me the power of web development. The quiz app is proof that with some effort and creativity, you can turn ideas into real projects.
